⚠️ Update Issue Fix
In the original release, the Update Button was incorrectly checking the original developer's repository for updates. Their latest changes have caused the app to break.
This has now been corrected, so updates will work properly going forward.

🔧 Already Updated and the App Broke?
Follow these steps to get back up and running:

Download twinvine_launcher-update-fix.py from this folder into your TwinVine-Launcher folder
Rename or delete the original twinvine_launcher.py file
Rename twinvine_launcher-update-fix.py to twinvine_launcher.py
Delete the TwinVine folder

⚠️ Before deleting — make sure to copy any video files you've already downloaded and your envied.yaml file


Relaunch the app and click Install TwinVine Tools

This should be quick, as it will check what tools are already installed



The app should then be working again. ✅

I've updated the original release, so this bug should no longer be an issue on a fresh install.
